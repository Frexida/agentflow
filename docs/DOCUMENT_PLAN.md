## 優先順位（案）
1. **Phase 1（v2開発前）**: プロジェクト管理計画書、WBS、機能要件定義書、画面設計書
2. **Phase 2（v2開発中）**: 業務フロー図、システム構成図、API仕様、ビルド手順書
3. **Phase 3（v2完成後）**: テスト関連、運用マニュアル、SLA
4. **Phase 4（事業化時）**: セキュリティ要件、インフラ構成、運用保守報告
