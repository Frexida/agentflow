# AgentFlow 料金プラン

## 概要

AgentFlow はフリーミアムモデルを採用。設計ツールは無料、クラウド機能は有料。

## 料金表

| プラン | 価格 | 内容 |
|--------|------|------|
| **Free** | $0 | ローカル保存 + 無制限 Export |
| **Pro** | $19/月 | クラウド保存 + テンプレート全開放 + バージョン履歴 |
| **Team** | $49/月 | 5席 + 共同編集 + 優先サポート |
| **Enterprise** | Custom | SSO + API + オンプレミス |

## 年払いオプション

| プラン | 月額換算 | 年額 | 割引率 |
|--------|----------|------|--------|
| Pro | $15/月 | $180/年 | 21% |
| Team | $39/月 | $468/年 | 20% |

## 価格設定の根拠

### 競合比較

| サービス | 価格帯 | AgentFlow との差別化 |
|----------|--------|---------------------|
| n8n Cloud | $24/月〜 | AgentFlow は設計ツール特化 |
| Dify | $59/月〜 | AgentFlow は OpenClaw 専用 |
| CrewAI | $99/月〜 | AgentFlow は実行コストなし |
| Flowise | $35/月〜 | AgentFlow は組織設計に特化 |

### $19 の理由

1. **$20 の壁** — 心理的に「試しやすい」価格帯
2. **実行コストなし** — 設計ツールなので低価格でも利益が出る
3. **OpenClaw ユーザー層** — 個人開発者が多く、高価格は刺さらない
4. **競合より安い** — n8n ($24)、Dify ($59) より低価格
